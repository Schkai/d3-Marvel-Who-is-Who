# d3-Marvel-Who-is-who
A d3.js project for university displaying Marvel characters and their relations based on Marvel's API.


### Get started:

* Download and unzip package
* `cd d3-marvel-who-is-who`
* `npm install` 
* `gulp`
* Access site on `localhost:8000`
